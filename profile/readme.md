## 🌽 What is Corn Studios?

Corn Studios is an independent software organization building practical, no-nonsense tools for Windows power users. We believe great software should be **free, open, and actually useful** — no bloat, no dark patterns, no paywalls.

Every project here is built with real use cases in mind and shipped with the community in focus.

## 💡 Philosophy

> Open source isn't just a license — it's a commitment to the people who use your software.

Corn Studios was founded on a simple idea: tools that help people shouldn't cost them anything. We're rooted in the spirit of the early open source community and we're keeping that alive, one project at a time.
